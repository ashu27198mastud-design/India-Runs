# RANKFORGE AI

**Beyond filtering. Evidence-backed candidate intelligence.**

Submission Link Requirement:
Please submit the public GitHub repository URL in the challenge portal. The repository must be publicly accessible.

## Problem Statement
Traditional ATS systems over-rely on keywords, missing hidden gems and ranking keyword-stuffed resumes too high. They lack contextual understanding, fail to explain their ranking decisions, and do not integrate career trajectory, project evidence, or activity signals into their evaluation.

## Product Name
**RANKFORGE AI** — Intelligent Candidate Discovery and Ranking Platform

## What Was Built
An intelligent candidate discovery and ranking system powered by Google GenAI. It transcends basic keyword matching by evaluating "Candidate DNA" and "Job DNA", scoring semantic fit, detecting overfit resumes, and identifying high-potential hidden gems.

## How the AI Works

### Ranking & Scoring Engine
The AI evaluates candidates across multiple dimensions:
* **Mandatory skill match**: Core competency matching.
* **Semantic fit**: Using `@google/genai` to understand context and adjacent skills.
* **Evidence Confidence**: Verifying claimed skills against concrete project evidence.
* **Career Trajectory & Activity Signals**: Evaluating growth rate and external signals (e.g., open source, certs).
* **Risk Adjustment**: Detecting overfit resumes (keyword stuffing with no evidence) or flight risks.

### Explainability & Responsible AI
Every candidate rank is supported by evidence, confidence, risk indicators, and a human-readable explanation generated by Gemini. 
* **Fairness Guardrail**: The system excludes protected attributes and bases rankings strictly on job-relevant synthetic candidate data.
* **Human Review**: AI acts as a decision support tool; human review is mandatory before final hiring decisions.

## Setup Instructions
1. Clone the repository.
2. Install dependencies: `npm install`
3. Set your Google AI Key in `.env`: `GEMINI_API_KEY=your_key_here`
4. Run locally: `npm run dev`

## Generating Ranked Output (PDF)
To generate the final competition output PDF, run:
\`\`\`bash
node scripts/generate-pdf.js
\`\`\`
This will output to `submission/ranked-candidates-output.pdf`.

## Demo Flow for Judges
1. Open the application dashboard (`http://localhost:3000`).
2. Recruiter enters/views the target job description (e.g., Senior GRC Analyst).
3. System instantly evaluates the candidate dataset, extracting Job and Candidate DNA.
4. View the AI-ranked candidates in the main table.
5. Identify the "Hidden Gem" and evaluate the "Overfit" candidate flags.
6. Click **Export PDF** to generate the final shortlisted output.

## Architecture
1. **Job Description Input** -> **Job DNA Extraction**
2. **Candidate Dataset** -> **Skill Ontology Matching**
3. **Google GenAI Ranking Engine** -> **Explainability Layer**
4. **Fairness Guardrail** -> **Ranked Output Generator**
